Based off your hourly wage, hours worked per day and days worked per week. This will calculate your:

1)Income tax
2)Income tax threshold
3)Tax rates
4)Daily income
5)Weekly income
6)Yearly income

I made this program from scratch in my first day of learning python. The idea was completely original and I had no help making it.


----------------------------------------------------------------------
How to install for Mac

1) Open terminal and type in "chmod +x income_tax_calculator.command"
2) Open the income_tax_calculator.command file
3) Wait for the program to open in terminal
4) Input the needed values
----------------------------------------------------------------------
How to install for Windows

1) Open terminal
2) Change directory to where the file is. eg.(cd desktop)
3) Type "python income_tax_calculator.py"
3) Input the needed values
